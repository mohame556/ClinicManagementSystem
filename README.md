# Clinic Management System

A comprehensive Clinic Management System built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

## Features

* User Authentication and Authorization
* Patient Management
* Doctor Management
* Appointment Scheduling
* Medical History Tracking
* Reports and Statistics
* User Roles Management
* Dashboard and Analytics

## Technologies Used

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* ASP.NET Identity
* Bootstrap
